 Accessibility Considerations 

The navigation bar is made to be responsive using Bootstrap; it should collapse into the hamburger menu at the medium breaking point for the best experience on tablet and mobile. To ensure accessibility b, use the <nav> element in every navbar to identify it as a landmark region for users using assistive technologies.

Usage

The navigation bar persiss across all pages.
